# W3C Community Group Form Draft

Status: Draft 0.2

Date: 2026-06-12

## Purpose

This file maps likely W3C Community Group proposal fields to the local draft
materials. The canonical copy-paste proposal text is maintained in
`docs/cws-html-w3c-cg-submission-copy.md`.

Keeping this file as a field map avoids maintaining the same scope, deliverable,
and status language in multiple places.

## Required Process Points

- Include the phrase: "This group may publish Specifications."
- Do not describe the work as a W3C Recommendation or W3C Standard.
- Be explicit that CWS HTML 1.0 Draft is an initial contribution, not a finished
  standard.
- The proposed group needs five individual supporters through the W3C proposed
  group page before creation.
- Supporting group creation should be presented as supporting a public venue,
  not endorsing every technical detail.

## Pre-Submission Check

Before copying text into the W3C form:

- all public links in `docs/cws-html-w3c-cg-submission-copy.md` point to the
  intended `main` branch URLs;
- public URLs work;
- license and contribution rights have been checked;
- evidence files are synthetic or cleared for publication;
- `npm run spec:validate` passes;
- `git diff --check` passes.
